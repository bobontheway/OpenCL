待办：
（1）理解设计模式是怎样解决问题的。
（2）设计模式可以帮助大家很快掌握那些经验丰富的开发人员才具有的知识和经验。
（3）可复用性、可替换性、接口（API）、继承和委托、抽象化等。
（4）对我们自己使用类库很有帮助。这是因为类库中的许多部分都与设计模式有关。
当设计类库时，设计模式的知识对我们的帮助会更大。
（5）不论是在什么领域，给“在某种场景下重复发生的问题的解决办法”赋予名字，并
整理而成的东西一般都被成为“模式”。设计模式是适用于软件设计和开发领域的模式，
它是模式中的一种。
（6）设计模式并非具体的实现，这些实现背后的思考方式和解决方法才是设计模式。
